## 1 Gradle 是什么
是一个开源的项目自动化构建, 建立在ant和maven基础上，并引入了Groovy的特定领域语言(DSL),不再使用XML形式管理构建脚本。
## 2 Groovy是什么
是用于Java虚拟机的动态语言。
